src.fin.juros.juros_compostos(c: float, cf: float, n: int) → float[código fonte]¶

Cálculo de juros compostos.

Calcula os juros compostos de um capital sobre o valor final.

Parâmetros

  • c (float) – Capital inicial

  • cf (float) – Capital atualizado no final dos períodos (Valor final).

  • n (int) – Número de perídos.

Retorna

Taxa de juros aplicada.

Tipo de retorno

float

Observação:

Essa obs tem como objetivo apenas demonstrar uma referencia cruzada para uma classe de um outro pacote src.scrap.draft.Calc, ou um método dessa classe: src.scrap.draft.Calc.times().

src.fin.juros.juros_simples(c: float, i: float, n: int) → float[código fonte]¶

Calculo de juros simples.

Calcula os juros simples do capital informado.

Parâmetros

  • c (float) – Capital sobre o qual será feito o cálculo.

  • i (float) – Taxa de juros por período.

  • n (int) – Número de períodos.

Retorna

Valor dos juros calculado sobre o capital, no perído informado.

Tipo de retorno

float
